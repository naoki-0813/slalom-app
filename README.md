# パイロンスラローム計測アプリ — Claude Code 開発スターター

スマートフォンを車に固定し、パイロンスラローム走行をセンサーで記録・分析する
Webアプリを **Claude Code で開発するための一式** です。

## 同梱物

```
pylon-slalom-starter/
├── README.md                 ← このファイル
├── docs/
│   └── 仕様書.md             ← アプリの要件定義・技術仕様
└── .claude/
    └── skills/
        └── pylon-slalom-dev/ ← Claude Code用の開発スキル
            ├── SKILL.md
            └── references/
                ├── sensors.md            (センサー・キャリブレーション)
                ├── video-and-storage.md  (動画録画・データ保存)
                ├── data-analysis.md      (分析・比較アルゴリズム)
                └── deployment.md         (公開方法・実機テスト)
```

## 使い方(3ステップ)

1. **このフォルダを開発用フォルダとしてPCに置く**
   (ZIPを展開した `pylon-slalom-starter` フォルダのままでOK。名前は変えても構いません)

2. **そのフォルダで Claude Code を起動する**
   ターミナル(コマンドプロンプト)でフォルダに移動して:
   ```
   cd pylon-slalom-starter
   claude
   ```

3. **最初の指示を出す**
   例えばこう入力します:
   ```
   docs/仕様書.md を読んで、フェーズ1(センサー記録+CSV)から開発を始めてください。
   私はIT初心者なので、少しずつ進めて説明しながらお願いします。
   ```

`.claude/skills/` に入っているスキルは、Claude Code がこのフォルダで作業するときに
自動で認識され、センサーの使い方やデプロイ手順などの詳細知識として参照されます。

## 開発の流れ(仕様書 §7 より)

1. **フェーズ1**: センサー記録 + CSV保存(まずPCの疑似データで動かす)
2. **フェーズ2**: 前方映像の録画・保存・グラフとの同期
3. **フェーズ3**: 熟練者との比較・再現性チェック
4. **フェーズ4**: 公開(GitHub Pages または AWS Amplify、ほぼ無料)して実機で使う

## 注意

- スマホでの実機テストには HTTPS の公開URLが必要です(手順はスキル内の deployment.md にあり、Claude Code が案内してくれます)
- クローズドコースでの使用を前提とし、運転中のスマホ操作はしないでください
